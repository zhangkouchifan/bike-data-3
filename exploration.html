This dataset includes ids of bike rides, the start - end time information, the member and rideable bike types. My main interests in this dataset is how the start time('started_at) relate to the bike usage. I think the total number, bike type('rideable_type') and user type('member_casual') might be the main features that helps the investigation.¶
